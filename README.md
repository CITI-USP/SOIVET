# SOIVET

Spatial Orientation in a Immersive Virtual Reality System (SOIVET) is an immersive system to assess the ability to translate allocentric to egocentric spatial reference frames (SOIVET Maze) and Route Learning abilities (SOIVET Route). This is the result from a collaboration between the Cognitive and Behavioral Neurology Group and the Center for Interdisciplinary Integrative Technologies from University of SÃ£o Paulo, Brazil.
